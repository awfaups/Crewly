# AI 成员自定义大模型 - 共识确认

## 已确认

- 在“新建 AI 成员”弹窗中增加模型配置区域。
- 模型配置包含提供商、模型名称、接入说明。
- 右侧 AI 成员卡片展示模型配置。
- 不保存密钥，界面明确提示密钥不应放入本地演示数据。

## 不做

- 不做真实模型调用。
- 不做 API Key 输入、保存或校验。
- 不做模型连通性测试。
- 不做按频道或任务切换模型。

## 验收口径

- 新建 AI 成员时可填写模型提供商和模型名称。
- 新建后右侧 AI 成员详情显示模型配置。
- 刷新页面后模型配置仍存在。
- 已有 AI 成员可正常显示默认模型配置。
- `npm run lint` 和 `npm run build` 通过。
