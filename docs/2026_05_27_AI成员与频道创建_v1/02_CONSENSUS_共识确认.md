# AI 成员与频道创建 - 共识确认

## 已确认

- 功能属于 Crewly 工作区新增能力。
- 继续沿用本地 mock 与 `localStorage` 持久化，不引入服务端。
- 创建入口放在左侧侧栏，靠近频道列表和 AI 成员列表。
- 新建频道后自动切换到该频道。
- 新建 AI 成员后自动选中该成员，并可在任务表单负责人下拉中选择。

## 不做

- 不做频道编辑、删除、归档。
- 不做 AI 成员编辑、删除、权限配置。
- 不做真实 Agent 能力配置。
- 不做跨设备同步。

## 验收口径

- 点击“新建频道”可打开中文表单并创建频道。
- 点击“新建 AI 成员”可打开中文表单并创建 AI 成员。
- 新对象刷新后仍存在。
- 新 AI 成员可作为任务负责人。
- `npm run lint` 和 `npm run build` 通过。
