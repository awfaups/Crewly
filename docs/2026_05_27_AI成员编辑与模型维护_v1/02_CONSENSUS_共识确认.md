# AI 成员编辑与模型维护 - 共识确认

## 已确认

- 右侧 AI 成员卡片增加编辑和停用入口。
- 编辑复用创建 AI 成员的表单。
- 保存后立即更新侧栏、右侧详情和任务负责人候选。
- 停用后从侧栏和新任务负责人下拉隐藏。
- 不删除历史任务、消息、会话中的成员引用。

## 不做

- 不做真实删除。
- 不做恢复入口。
- 不做 API Key 管理。
- 不做模型连通性测试。

## 验收口径

- 可从右侧 AI 成员卡片打开编辑弹窗。
- 可保存名称、职责、头像和模型配置。
- 停用后 AI 成员从侧栏隐藏。
- 新建任务负责人下拉不再展示停用成员。
- 刷新后编辑和停用结果保留。
- `npm run lint` 和 `npm run build` 通过。
