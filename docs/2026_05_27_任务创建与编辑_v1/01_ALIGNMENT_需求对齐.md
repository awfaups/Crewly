# 任务创建与编辑 - 需求对齐

## 目标

让 Crewly 的任务不再只来自 mock 数据，支持在工作台中创建、编辑、归档，并从频道消息转成任务。

## 范围

纳入：

- 新建任务。
- 编辑任务标题、描述、优先级、标签、截止时间、负责人。
- 从频道消息一键转任务。
- 归档任务。
- 任务和 Session 自动建立关联。
- localStorage 持久化。

不纳入：

- 后端 API。
- 多人协作冲突。
- 权限模型。
- 复杂表单校验。

## 验收标准

- 新建任务后出现在任务看板。
- 编辑任务后右侧详情和看板同步。
- 消息转任务后消息卡出现任务引用。
- 归档任务后从看板隐藏。
- 刷新页面后保留任务变更。
