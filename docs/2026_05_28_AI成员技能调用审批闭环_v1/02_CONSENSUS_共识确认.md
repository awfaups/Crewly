# AI 成员技能调用审批闭环 - 共识确认

## 共识

- 技能调用是模拟行为，只记录运行轨迹和审批状态。
- 调用入口放在右侧 AI 成员详情，和当前任务、当前 Session 绑定。
- 已安装技能才允许调用。
- 高风险技能或外部动作需审批时，进入审批流。
- 审批结果继续沿用现有通过/拒绝按钮。

## 验收标准

- 当前 AI 成员详情展示“可调用技能”。
- 点击低风险技能后，Agent Session 增加工具调用和模拟结果。
- 点击高风险技能后，Agent Session 增加等待审批事件，并出现审批卡。
- 审批通过/拒绝后，Agent Session 追加对应结果事件。
- 刷新后调用事件和审批状态仍保留。
