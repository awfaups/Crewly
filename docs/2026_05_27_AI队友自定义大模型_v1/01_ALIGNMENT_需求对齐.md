# AI 队友自定义大模型 - 需求对齐

## 背景

当前 Crewly 已支持创建 AI 队友，但 AI 队友只有名称、职责和头像。用户希望新增 AI 队友时可以配置其使用的大模型。

## 目标

- 新增 AI 队友表单支持大模型配置。
- 配置随 AI 队友一起持久化到本地演示状态。
- 右侧 AI 队友详情能看到该队友的大模型配置。
- 不保存 API Key、Token 或其他敏感凭据。

## 范围

- 前端演示级配置，不发起真实模型调用。
- 继续使用 `localStorage` key `crewly.workspace.v1`。
- 支持提供商、模型名称和接入说明。
- 兼容已有成员数据，旧 AI 队友默认补齐模型配置。

## 假设

- “自定义大模型”本阶段指模型配置元数据，不包含真实密钥管理。
- API Key 后续应放到后端或安全配置，不写入浏览器 localStorage。
- 预置提供商包括 OpenAI、Anthropic、DeepSeek、通义千问、智谱 AI、Ollama 和自定义。
