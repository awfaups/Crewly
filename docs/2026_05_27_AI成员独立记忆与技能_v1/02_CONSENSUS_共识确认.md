# AI 成员独立记忆与技能 - 共识确认

## 已确认

- 独立配置粒度是“每个 AI 成员”，不是全局共享。
- 记忆、技能和技能调用策略都挂在 `Member` 上。
- 创建和编辑 AI 成员时可维护这些配置。
- 管理模块展示每个成员的独立记忆和技能摘要。
- 当前阶段只做配置，不做真实执行。

## 不做

- 不接向量数据库。
- 不执行真实技能调用。
- 不做技能市场安装流程。
- 不做权限审计日志。

## 验收口径

- 新建/编辑 AI 成员弹窗包含独立记忆配置。
- 新建/编辑 AI 成员弹窗包含技能安装和调用策略。
- 管理模块能看到每个 AI 成员的记忆和技能摘要。
- 刷新后配置保留。
- `npm run lint` 和 `npm run build` 通过。
