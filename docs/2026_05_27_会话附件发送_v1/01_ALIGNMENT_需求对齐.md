# 会话附件发送 - 需求对齐

## 目标

让 Crewly 频道会话支持发送图片与文件，使 demo 更接近日常团队协作场景。

## 范围

纳入：

- Composer 选择图片或文件。
- 消息携带附件。
- 图片附件显示缩略预览。
- 文件附件显示文件名、大小和下载入口。
- 附件随消息写入 localStorage。

不纳入：

- 真实文件上传服务。
- 文件权限和病毒扫描。
- 大文件分片上传。
- 云端对象存储。

## 验收标准

- 选择图片后能发送，并在消息流展示图片预览。
- 选择非图片文件后能发送，并展示文件信息。
- 附件消息刷新后仍保留。
- 空文本但有附件时允许发送。
- 无文本且无附件时不可发送。
