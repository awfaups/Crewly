# AI 成员独立管理模块 - 共识确认

## 已确认

- AI 成员管理作为独立模块呈现。
- 模块入口放在工作台主区域顶部导航和左侧快捷入口中。
- 管理模块展示全部未停用 AI 成员。
- 管理模块内可以新建、编辑、停用 AI 成员。
- 停用成员继续保留历史引用。

## 不做

- 不新增 Next.js 路由。
- 不做真实模型连接测试。
- 不做 API Key 明文输入。
- 不做停用恢复。

## 验收口径

- 可从导航进入 AI 成员管理模块。
- 管理模块显示 AI 成员列表和模型配置摘要。
- 管理模块可新建、编辑、停用 AI 成员。
- 返回工作台后原有消息、任务、右侧详情仍可用。
- `npm run lint` 和 `npm run build` 通过。
