# 大模型真实配置场景 - 共识确认

## 已确认

- 模型配置应包含真实接入常用字段。
- 当前阶段仍是前端配置，不执行调用。
- 新建 AI 队友时需要填写完整模型配置。
- 右侧 AI 队友卡片展示关键配置。
- 旧配置自动迁移为新版配置。

## 不做

- 不保存 API Key 明文。
- 不做模型连通性测试。
- 不做服务端代理实现。
- 不做不同 provider 的实时模型列表拉取。

## 验收口径

- 新建 AI 队友表单包含 Base URL、认证方式、密钥引用名、运行环境、参数和能力开关。
- 新建后右侧能看到完整关键配置。
- 刷新后配置仍保留。
- 旧模型配置不会导致页面异常。
- `npm run lint` 和 `npm run build` 通过。
